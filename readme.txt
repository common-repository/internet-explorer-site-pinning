== Description ==

You can check for more information <a href="http://www.pedrolamas.com/projectos/wordpress-internet-explorer-site-pinning-plugin/">here</a>.

== Installation ==

1. Download the file and unzip. If the folder `internet-explorer-site-pinning` was not created, create a new directory called that and add the extracted files to it.
1. Upload `internet-explorer-site-pinning` folder to the `/wp-content/plugins/` folder on your web server
1. Activate the plugin through the 'Plugins' menu in WordPress
